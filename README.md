# @flowtr/fdpl

A hosting control panel written with Typescript, tinyhttp, & ESM.

## Features

- Continuous Integration With Customizable Presets (uses docker)
- Web UI to manage various services
- Dynamic database models (CMS)
- OpenAPI v3 schema
